About the Semantical Web

The Semantical Web is an extension of the World Wide Web through standards set by the World Wide Web Consortium (W3C). The goal of the Semantical Web is to make Internet data machine-readable.

By using ontologies, the Semantical Web allows data to be shared and reused across application, enterprise, and community boundaries. It is a collaborative effort led by W3C with participation from a large number of researchers and industrial partners.
